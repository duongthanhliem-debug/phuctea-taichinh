Lấy tại console.anthropic.com — dùng cho AI đọc ảnh. Nếu đã có Apps Script URL, bỏ qua mục này (proxy qua Apps Script không bị CORS).
💡 Apps Script proxy (khuyên dùng): Thêm đoạn sau vào Apps Script của bạn để AI đọc ảnh không bị lỗi CORS:
case 'claudeVision': var r=UrlFetchApp.fetch('https://api.anthropic.com/v1/messages',{method:'post',headers:{'x-api-key':e.anthropicPayload?e.apiKey||PropertiesService.getScriptProperties().getProperty('ANTHROPIC_KEY'):'','anthropic-version':'2023-06-01','Content-Type':'application/json'},payload:JSON.stringify(e.anthropicPayload),muteHttpExceptions:true}); return ContentService.createTextOutput(JSON.stringify({result:JSON.parse(r.getContentText()).content[0].text})).setMimeType(ContentService.MimeType.JSON); Hoặc lưu API Key vào Script Properties với tên ANTHROPIC_KEY.
